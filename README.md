# Local Setup

- Create a virutal environment and activate the VE
- install packages using pip install -r requirement.txt command
- Create a database named 'appdb' locally. 
- In the config.py file, update your local postgres password. Update the DATABASE_URL to 
``` postgresql+pg8000://user(postgres in my case):yourpassword@localhost:5432/appdb'```

Update DATABASE to : ``` postgresql://user(postgres in my case):yourpassword@localhost:5432/appdb ```
